HTML Opdracht: Match app

Je maakt individueel een matching HTML toepassing voor het vinden van ... een nieuwe hobby, nieuw huisdier, ...

Je gebruikt binnen de folder (voornaam_achternaam-matchapp) het HTML document (index.html) en images folder. In het HTML document heeft Emmet je de basisstructuur ("!" of "doc") gegeven waarin je de matching applicatie zal maken, door minstens gebruik te maken van de onderstaande elementen en attributen. Wanneer je alle checkboxes (selectievakjes) hebt aangevinkt heb je de volledige score van de opdracht behaald (30).
