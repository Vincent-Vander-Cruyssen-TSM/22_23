HTML Opdracht: Artikel

Verdeel de inhoud van een artikel (wiki, nieuws,...) a.d.h.v. de onderstaande HTML elementen.
Je maakt binnen een nieuwe folder (voornaam_achternaam-artikel) een HTML document (index.html) en images folder. In het nieuwe HTML document geeft Emmet je de basisstructuur ("!" of "doc") waarin je de inhoud (van het artikel) correct zal verdelen, door minstens gebruik te maken van de onderstaande elementen en attributen. Wanneer je alle checkboxes (selectievakjes) hebt aangevinkt heb je de volledige score van de opdracht behaald.
