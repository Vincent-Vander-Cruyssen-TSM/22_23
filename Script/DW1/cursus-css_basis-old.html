CSS

Alle HTML elementen hebben standaard stijlkenmerken. Wanneer je een eigen website maakt, kun je deze stijlkenmerken aanpassen met CSS. Je kunt dan informatie toevoegen aan HTML-elementen, zoals kleur, lettertype, afstand en achtergrond. Dit kan binnen het document zelf (als attribute van ieders element of met een <style>-element) of [nog beter] in een extern CSS-bestand aan de hand van het <link>-element.

In het kort...

CSS staat voor Cascading Style Sheets. Het is een taal die wordt gebruikt om het uiterlijk, of lay-out, van een webpagina te beschrijven. CSS wordt gebruikt om alle HTML-elementen op een pagina op te maken, inclusief de koppen, alinea's en andere stukken tekst. Je kunt met CSS onder andere elementen op de pagina positioneren, kleuren toewijzen en de achtergrondafbeelding bepalen. CSS kan niet alleen worden gebruikt om de lay-out van een pagina op te maken, maar ook om functies zoals animaties en vervolgkeuzemenu's (drop-down menu's) toe te voegen.

Waarom Cascading Style Sheets?

Omdat het een stijlblad is dat verschillende stijlen toepast op een document, waarbij de stijlen van boven naar beneden worden toegepast (van cascade: waterval).

CSS Vernieuwingen

CSS is de afgelopen jaren steeds complexer geworden en kan inmiddels veel meer dan vroeger. Met het nieuwe zogenaamde flex en grid systeem kunnen we eenvoudig vernuftige layout-constructies maken. Deze layouts zijn responsief, werken soepel en zijn op alle schermgroottes goed te zien, van smartphone tot tv.

CSS grid is een tweedimensionale lay-outmethode die een gebied in rijen en kolommen verdeelt. Deze methode is zeer flexibel, zodat we alles kunnen aanpassen aan elk scherm.

CSS flex is een een-dimensionale lay-outmethode die werkt met flex-items. Deze flex-items kunnen in een rij of kolom gestapeld worden. Dit is afhankelijk van de flex-richting. Deze methode is ook zeer flexibel, zodat we de flex-items in verschillende richtingen kunnen neerzetten.

CSS Declaration (verklaring)

Een CSS-regelset bestaat uit een selector en een declaratieblok. De selector wijst naar het HTML-element dat u wilt opmaken. Het declaratieblok bevat een of meer declaraties, gescheiden door puntkomma's. Elke declaratie bevat een CSS-eigenschapsnaam en een waarde, gescheiden door een dubbele punt.

p {   color: red;   text-align: center; }

CSS-declaraties eindigen altijd met een puntkomma en declaratieblokken zijn omgeven door accolades. De volgorde van de declaratie is belangrijk. Wanneer meerdere declaraties dezelfde eigenschap hebben, wordt alleen de laatste gebruikt.

p {   color: red;   color: blue; (deze wordt gebruikt) }

CSS Selectors

Met CSS maak je gebruik van verschillende selectors. Hiermee selecteer je de elementen waarop je stijl wilt toepassen. De selectors kunnen verschillende vormen aannemen, afhankelijk van de elementen waarop je wilt selecteren.

  • Een elementselector: selecteert alle elementen van een bepaalde soort.
  • Een id-selector: selecteert een element met een bepaald id.
  • Een klasseselector: selecteert alle elementen met een bepaalde class.
  • Een pseudo-selector: selecteert elementen op basis van hun status of positie.

Elementselector

De eenvoudigste selector is een elementselector. Met deze selector selecteer je alle elementen van een bepaald soort. Je gebruikt hiervoor de naam van het element. Denk bijvoorbeeld aan de HTML-variabele `<h1>` (kop 1).

h1 {   property (eigenschap): value (waarde); }

Id-selector

Een id-selector in CSS is een aanduiding waarmee je kunt opgeven welke stijlen worden aangepast voor een element met een bepaalde id-waarde. Een id-selector wordt geschreven als een hashtag (#) gevolgd door de id van het element. In het onderstaande voorbeeld wordt de id #uniek gebruikt om de stijlen van het element met id="uniek" toe te passen.

#uniek {   background-color: yellow; }

Klasseselector

Een klasseselector in CSS is een aanduiding waarmee je kunt opgeven welke stijlen worden toegepast op een HTML-element met een class-attribute. Een klasseselector is een aanduiding die begint met een punt (.).In de onderstaande CSS-regel wordt de klasseselector ".groepje" gebruikt. Dit betekent dat de stijlen worden toegepast op alle HTML-elementen met de klasse "groepje".

.groepje {   background-color: red; }

Een pseudo-selector

Een pseudo-selector is een aanwijzing waarmee je kunt opgeven of je een bepaalde stijl wilt toepassen op een element dat in een bepaalde relatie tot een ander element staat. Een voorbeeld van een pseudo-selector is :hover, waarmee je kunt opgeven dat je een bepaalde stijl wilt toepassen op een element wanneer de muisaanwijzer eroverheen gaat.

a:hover {   color: red; }

CSS eigenschappen

CSS eigenschappen of properties zijn de stylingsmogelijkheden die je hebt binnen CSS. Ze bepalen hoe je HTML-elementen kunt opmaken, zoals de kleur, de lettertype, de achtergrond, de margins en de padding.

Shorthands

Een shorthand is een manier om een aantal CSS-regels in één regel te schrijven. Het is een kortere manier om dezelfde CSS-regels te schrijven. Het is beter om de shorthand-methode in sommige gevallen te vermijden omdat de code moeilijker te lezen wordt. Voor margin en padding is het aan te raden om de shorthand wel te gebruiken.

background eigenschap

De background eigenschap in CSS is een kortere manier om achtergrondkleur, achtergrondafbeelding, achtergrondherhaling, achtergrondpositie in één regel te definiëren. Gemakkelijker is het vaak om deze eigenschappen afzonderlijk te definiëren.

body {  background-color: #ff0; }

color eigenschap

De color eigenschap in CSS bepaalt de kleur van de tekst in een element. De kleur kan worden opgegeven als een hexadecimaal getal, RGB- of HSL-waarde, of een naam van een standaard kleur.

p { color: red; color: #ff0000; color: rgb(255,0,0); color: hsl(0,100%,50%); }

Additieve kleuren

Additieve kleuren zijn kleuren die door licht te combineren worden gevormd. Het licht dat we zien is een mix van verschillende kleuren licht, elk met hun eigen golflengte. Wanneer deze kleuren licht samenkomen, worden ze toegevoegd aan elkaar en wordt het totaal licht dat we zien helderder. De primaire additieve kleuren zijn rood, groen en blauw. Wanneer deze kleuren licht in gelijke verhoudingen mixen, krijg je wit licht.
