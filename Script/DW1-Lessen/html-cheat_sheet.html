Main root

<html> … </html>

Het HTML <html>-element vertegenwoordigt het root (element op het hoogste niveau) van een HTML-document, dus het wordt ook wel het wortelelement genoemd. Alle andere elementen moeten afstammelingen van dit element zijn.
