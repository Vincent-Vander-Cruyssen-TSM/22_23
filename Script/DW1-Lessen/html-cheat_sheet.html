HTML Cheat Sheet

Main root

<html> . . . </html>

Het <html> element vertegenwoordigt de root, element op het hoogste niveau, van een HTML-document, dus het wordt ook wel het root element genoemd. Alle andere elementen moeten afstammelingen van dit element zijn.

<!DOCTYPE html>
<html lang="en">
<head> ... </head>
<body> ... </body>

Document metadata

<head> . . . </head>

Het <head> element bevat informatie (metadata) over het document, zoals de titel, scripts en stylesheets.

<link>

Het <link>, of External Resource Link, element specificeert relaties tussen het huidige document en een externe bron. Dit element wordt meestal gebruikt om te linken naar stylesheets, maar wordt ook gebruikt om sitepictogrammen vast te stellen.
